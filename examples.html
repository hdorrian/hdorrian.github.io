Example 1: Formulas

If you’re asked how many days in a week there are, you would find it easy 7. How about the number of days in two weeks? Again easy, there’s 14.

How about how many days there are in three weeks, four weeks or five weeks? If you know the formulae, its simple;

  • 1 week is 1x7 days = 7 days
  • 2 weeks is 2x7 days = 14 days
  • 3 weeks is 3x7 days = 21 days
  • 4 weeks is 4x7 days = 28 days

So, the formula to work out the number of days is:

Number of days = number of weeks x 7

Questions

  1. How many pence are there in £1?
  2. How many pence are there in £7?
  3. Write down a formula connecting the number of pence to the number of pounds.

Answers

  1. £1 = 100p
  2. £7 = 700p
  3. Pence = Pounds x 100

Example 2: Sequences

Cube Numbers

Cube numbers are formed by multiplying a digit by itself three times;

  • 1 cubed is 1x1x1 = 1
  • 2 cubed is 2x2x2 = 8
  • 3 cubed is 3x3x3 = 27

Triangle Numbers

The triangle numbers are:

  • 1 triangled is 1+2 = 3
  • 2 triangled is 1+2+3 = 6
  • 3 triangled is 1+2+3+4 = 10

Example 3: Numbers

Even Numbers

  • Any number that can be divided by 2 is called an even number
  • Some examples of even numbers are 2, 4, 6, 8, 22, 144, 40012
  • If a number ends with an even digit, then it is an even number

Odd Numbers

  • Any number that cannot be divided by 2 is called an odd number
  • Some examples of odd numbers are 1, 3, 5, 7, 9, 177, 3895
  • If a number ends with an odd digit, then it is an odd number
